## 📂 Repository Structure

The solutions in this repository are organized automatically by problem. Each directory generally includes:
* **`README.md`**: The complete problem statement, including constraints and examples.
* **Solution File**: The accepted code (e.g., `.py`, `.cpp`).
* **Complexity Notes**: Automated notes on Time and Space Complexity based on the submission.
